## 这是什么？

这是一个基于 Claude Code 的 AI Skill，提炼了作家**江南（杨治）**的完整思维框架。江南是中国当代奇幻文学的代表人物，代表作包括《龙族》系列（累计发行超2100万册）、《九州缥缈录》系列、《此间的少年》、《上海堡垒》等。

本 Skill 通过深度调研（40+信息源，一手来源>60%），提炼出：
- **6个核心心智模型**：衰小孩视角、糖衣苦药双层叙事、画面即故事影像思维、孤独炼金术情感转化、名场面引爆节奏控制、人物反推框架迭代创作
- **8条决策启发式**：从"求而不得的故事核"到"能用对话解决就不叙述"
- **完整的表达DNA**：句式特征、修辞体系、高频词汇、幽默方式
- **智识谱系与诚实边界**

## 能做什么？

- 🖊️ **文学创作分析**：用江南的心智模型分析叙事设计、人物塑造、世界观构建
- 🎬 **IP商业化评估**：从"衰小孩视角"和"糖衣苦药"框架评估作品的IP潜力
- 🧠 **创作者困境诊断**：理解"断更"、"烂尾"、"燃烧vs持久"的结构性原因
- 🎯 **思维顾问**：用江南的6个镜片分析商业决策、创意方向、职业选择
